# About

This project is a web application developed using vanilla HTML, CSS and JavaScript and XAMPP as the backend.

The project was built to fulfill Web Application Development project assignment and is made by TC2L Group 3

# How to Setup
## Windows Pre-requisite
- In order to run this project on your local machine, you need to have XAMPP installed on your local machine. 
- Run XAMPP and ensure that MySQL Database and Apache Web Server are running

## Setup
1. Clone this repository onto your local machine.
2. Copy the project file into the "htdocs/" directory of your XAMPP files.
3. In the PHPMyAdmin page, import "cashier_schema.sql" from the database directory into the MySQL database.
4. Navigate to "localhost/gaming-store-webapp/public" to be taken to the home page.
